# NLP_Python
Spelling correction by Python

How to use framework
- download:
  python -m pip install pyqt5
  python -m pip install pyqt5-tool
  
- Convert file .ui to file .py
  - Open cmd into folder project
  - pyuic5 -x "path file .ui" -o "path file .py"
